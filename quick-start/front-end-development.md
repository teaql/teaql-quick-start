
## 前端开发 ##

**准备工作**

1. git的账号创建  



**在git里创建一个repository**

1. 当你进入成功创建Git账号并进入后, 点击左上角的**Create repository**创建一个repositoty
![](images/RC-1.png)

2. 跳转到另一个新的界面  
    + 先给新建的repository起一个名字 （例如elimy-notebook） 
    + 记得点击*Add a README file* 绿色按键 
    + 最后点击Create repository  绿色按键   
    
![](images/RC-2.png)




**在Terminal里打开**

1. 粘贴并进入
    点击*Code* 绿色按键之后会弹出一个新的小窗口点击复制键（如红色框所圈） 
    
![](images/RC-3.png)


2. 打开Terminal
